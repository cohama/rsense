RSense
======

Documentation
-------------
easy to use RSense in Vim with NeoBundle

Installation
------------
```
NeoBundle 'cohama/rsense', {
      \     'build': {
      \        'unix': '/bin/sh install.sh'
      \      }
      \    }
```
